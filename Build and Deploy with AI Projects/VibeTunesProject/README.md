# VibeTunes 

A sleek, modern music player built with **HTML, CSS, and Vanilla JavaScript**.  
VibeTunes fetches real music data from Deezer and allows users to preview tracks with a clean, responsive UI.

**Technical Spec:**  
[spec.md](./spec.md)

---

## Features
- Search music by genre or keyword
- Play, pause, and skip tracks
- Audio preview playback
- Progress bar with timestamps
- Top Tracks carousel
- Dark theme with glowing UI
- Mobile-friendly layout

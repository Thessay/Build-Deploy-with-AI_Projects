# Sticky Todo App

A clean and minimal todo list app with sticky-note cards, built using pure HTML, CSS, and JavaScript.

 **Technical Spec:**  
[spec.md](./spec.md)

---

## Features
- Add, complete, delete todos
- Filters (All / Active / Completed)
- Clear completed tasks
- Sticky-note UI with rotation and shadow
- Light / Dark mode
- Keyboard shortcuts
- Persistent storage
